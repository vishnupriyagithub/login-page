<h1>Login Page</h1>

 

<h2>Description</h2>
Project consists of a simple userlogin page and signin page where the user can enter username or email, password. This code is written in HTML , CSS and JavaScript.I'm still working on it to link it to Firebase to store realtime user data.
<br />


<h2>Languages and Utilities Used</h2>

- <b>HTML5</b> 
- <b>CSS</b>
- <b>JAVA SCRIPT</b>

<h2>Environments Used </h2>
